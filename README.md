# RTSP/RTP Video Streaming Application

## Overview
A client-server video streaming application built using RTSP for control and RTP for real-time data transmission. The client sends playback commands while the server streams video frames over the network.

## Features
- Client-server architecture using sockets  
- Playback controls: Setup, Play, Pause, Teardown  
- Real-time MJPEG video streaming over UDP  
- GUI-based video display  

## Tech Stack
Python, Sockets (TCP/UDP), RTSP, RTP

## Summary
This project demonstrates the basics of real-time streaming, including protocol-based communication and packetized video transmission.
